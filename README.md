# 😉 Student Management Program - System
학생 통합 관리 시스템


## 🖥️ 프로젝트 소개
학생 통합 관리는 학생들의 수강 정보와 과목, 학과를 통합하여 관리합니다.
학생이 수강 신청을 실수하여 잘못 신청하여도 통합 관리를 통해,
실수한 부분을 관리자가 바로 수정 할 수 있도록 하는 프로그램 입니다.

## 📌 주요 기능

### 👤 학생_관리

관리자가 '학생 목록' 으로 비밀번호와 주소를 제외한 나머지를 볼 수 있다.

'학생 등록' 은 학생의 학과가 선택됨과 동시에 학번은 자동적으로 입력이 되고,
그외 나머지 정보들은 관리자가 직접 입력하여 해당 학과에 인원을 등록 및 추가한다.

'학생 수정' 해당 학생의 학번을 입력하여 해당 학생의 정보를 수정한다.

'학생 삭제' 해당 학생의 학번을 입력하여 해당 학생의 정보를 모두 삭제한다.

'학생 검색' 학생의 이름 한 글자만 입력을 해도 입력한 한 글자만 해당되면 모두 찾는다.

![image](https://github.com/user-attachments/assets/57ed1d80-71c7-480f-8f61-1ccda300d275)

### 📚 과목_관리

관리자가 '과목 목록' 을 일련번호로 관리하고 과목번호와 과목명을 보여준다.

'과목 등록' 과목 번호로 등록을 하고, 과목명을 추가한다.

'과목 수정' 해당 과목을 수정할때는 일련번호로 관리하고 과목번호와 과목명을 명명한다.

'과목 삭제' 해당 과목을 삭제할때는 일련번호를 입력하여 해당 과목을 삭제한다.

'과목 검색' 과목명의 한글자만 입력 하더라도 해당 과목명의 모두를 보여준다.

![image](https://github.com/user-attachments/assets/3840a4b4-2138-44ec-8820-d5bc8671e8e6)

### 🏫 학과_관리

-관리자가 '학과 목록' 을 순번과 약어 학과명으로 관리한다.

'학과 등록' 학과 약어로 등록을 하고, 학과명을 추가한다.

'학과 수정' 해당 학과를 수정할때는 순번으로 관리하고 학과의 약어와 학과명을 명명한다.

'학과 삭제' 해당 학과을 삭제할때는 순번을 입력하여 해당 학과를 삭제한다.

'학과 검색' 헉과명의 한글자만 입력 하더라도 해당 학과명의 모두를 보여준다.

![image](https://github.com/user-attachments/assets/3b9da464-a3bb-4b1f-8426-b74752aab52e)

### 🏫 수강_관리

-관리자가 '수강 신청목록' 은 순번과 약어 학과명으로 관리한다.

'수강 등록' 수강을 등록 할 수 있도록 함

'수강 취소' 신청목록에 있는 수강을 순번으로 취소를 할 수 있도록 함
